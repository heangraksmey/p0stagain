🚨 TROUBLESHOOTING:

   Problem: "Server Not Connected"
   Solution: Run "npm start" in terminal

   Problem: "Channel not found"
   Solution: Use @channelname or numeric ID

   Problem: "Dashboard won't load"
   Solution: Refresh page, check server running

   Problem: "Can't find a file"
   Solution: All files in same directory, double-check name

   → See GET_STARTED.md for more solutions
