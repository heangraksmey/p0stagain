# 🚀 Get Started in 3 Steps!

## Step 1: Start the Backend Server

Open your terminal and run:
```bash
npm install
npm start
```

You should see:
```
🤖 Telegram Bot Server running on port 3000
```

**Leave this running!** ✅

---

## Step 2: Open the Dashboard

Choose ONE of these options:

### Option A: HTML Dashboard (Easiest - No React Needed!) ⭐
1. Open `telegram-bot-dashboard.html` in your browser
2. Done! The dashboard loads immediately

### Option B: React Dashboard
1. Copy `TelegramBotDashboard-improved.jsx` to your React app
2. Or paste it into Claude.ai as a React artifact

---

## Step 3: Add a Channel & Post!

1. **Go to "Channels" tab**
2. **Add your first channel:**
   - Channel ID: `@your_channel_name` (or numeric ID)
   - Name: `My Channel`
   - Click "Add Channel"

3. **Go to "Post Message" tab**
4. **Write and send:**
   - Select your channel
   - Type a message
   - Click "Send Message"

✅ **Done!** Your message is on Telegram!

## 🎯 Features to Try

### Send Without Sound 🔇
- Toggle "Silent" 
- Message posts without notification

### Hide with Spoiler 👁️
- Toggle "Spoiler"
- Message is hidden until users tap it

### Schedule Message ⏰
- Pick a date/time
- Message sends automatically
- Cancel before it sends

### Broadcast to Multiple Channels 📢
- Toggle "Broadcast Mode"
- Select multiple channels
- Send same message to all

## 🔧 Troubleshooting

### "Server Not Connected"
❌ Make sure `npm start` is running in terminal
✅ Check for errors in terminal

### "Channel not found"
❌ Double-check your channel ID
✅ Use `@channelname` for public channels
✅ Use numeric ID for private channels

### Dashboard won't load
❌ Make sure server is running
✅ Try refreshing the page (Ctrl+R)
✅ Clear browser cache
